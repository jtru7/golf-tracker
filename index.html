Find this in your Google Sheets URL: https://docs.google.com/spreadsheets/d/SPREADSHEET_ID/edit

Setup Instructions

  1. Create a new Google Sheet
  2. Go to Google Cloud Console
  3. Create a new project or select existing
  4. Enable Google Sheets API
  5. Create credentials (API Key)
  6. Restrict the API key to Sheets API only
  7. Copy the API key and spreadsheet ID above
  8. Make your spreadsheet publicly viewable (or use OAuth)
