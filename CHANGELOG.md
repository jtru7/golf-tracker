## [0.1.0] - 2025-02-07

### Added
- Initial project scaffolding with separated HTML, CSS, and JS files
- Hole-by-hole round entry with score, putts, penalties, fairway direction, approach result, and sand saves
- Dashboard with handicap index (USGA), avg score, FIR%, GIR%, putts/round, and scrambling %
- Course management (add, edit, delete) with per-hole par and yardage
- Support for 9-hole and 18-hole courses
- Round filtering by date range and count (last 5/10/20)
- JSON data export
- localStorage persistence
- Test course (TRU TEST Course, 9 holes, Rexburg ID)
